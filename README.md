# panel-nfl-journal

本仓库用于归档和发布多个独立 HTML 页面，主要用于记录和展示与 NFL 赛事相关的信息面板与日志内容。

## 项目简介

panel-nfl-journal 是一个静态页面集合仓库，聚焦于 NFL 赛事数据的可视化与信息呈现。每个页面独立运行，不依赖后端服务，适合通过 GitHub Pages 或类似平台直接部署。

## 目录说明

```
panel-nfl-journal/
├── index.html          # 主入口页面（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放所有归档的 HTML 页面，每个文件对应一个独立面板或日志。
- `assets/` 目录用于存放页面所需的公共样式、脚本及图片资源。

## 页面归档说明

仓库中收录的 HTML 页面均为独立归档，内容涵盖赛事数据展示、信息面板、日志记录等形式。每个页面均可在浏览器中直接打开，无需额外配置。

- 页面之间互不依赖，可单独使用。
- 资源文件（CSS/JS）建议引用相对路径，便于本地或在线部署。
- 部分页面可能引用外部公共 CDN 库（如 Chart.js、D3.js 等），请确保网络可访问。

## 维护说明

本仓库由个人或团队维护，用于长期归档与更新 NFL 相关数据面板。欢迎通过 Issues 提交建议或报告问题。

- 新增页面请放入 `pages/` 目录，并保持命名规范。
- 如有公共资源更新，请同步修改 `assets/` 下对应文件。
- 避免在页面中引入非必要的外部链接或敏感内容。
